# UNIX/Bash navigation and commands

### Common UNIX/Bash commands

#### cat 
reads the output to the screen 
```
cat file.txt
```

#### mv
Moves one file to another file
```bash
mv file.txt ../
```

### ls
Lists the contents of the current folder
```bash
ls
> file.txt
```
